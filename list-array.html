Lista/array de ovejas

6. Considere una lista/array de ovejas. Cada oveja tiene un nombre y un color. Cree una función que devuelva una lista de todas las ovejas rojas cuyos nombres también contengan las letras n y a, sin importar el orden, las mayúsculas o los espacios.Recuerda. Debe contener las dos letras "a" y "n" en el nombre. No cuente ovejas con una sola letra, debe tener dos letras.
